  <name>Task 1: Implement RoomCanvas.tsx with StrictMode-safe rAF loop</name>
  <files>src/RoomCanvas.tsx</files>
  <action>
    Create `src/RoomCanvas.tsx`. This file is allowed to import React and use JSX.

    Props interface:
    ```typescript
    interface RoomCanvasProps {
      heightmap: string;  // Habbo heightmap string — triggers re-render on change
    }
    ```

    Refs (all useRef — NO useState for any render-loop value):
    ```typescript
    const canvasRef   = useRef<HTMLCanvasElement>(null);
    const runningRef  = useRef(false);
    const rafIdRef    = useRef(0);
    const renderState = useRef<{
      offscreenCanvas: OffscreenCanvas | null;
      cameraOrigin:    { x: number; y: number };
      mainCtx:         CanvasRenderingContext2D | null;
    }>({
      offscreenCanvas: null,
      cameraOrigin:    { x: 0, y: 0 },
      mainCtx:         null,
    });
    ```

    useEffect with [heightmap] dependency (runs on mount AND when heightmap prop changes):
    Step 1: Get canvas from canvasRef.current — return early if null.
    Step 2: Call initCanvas(canvas) -> mainCtx; store in renderState.current.mainCtx.
    Step 3: Parse the heightmap: const grid = parseHeightmap(heightmap).
    Step 4: Compute camera origin: renderState.current.cameraOrigin = computeCameraOrigin(grid, canvas.offsetWidth, canvas.offsetHeight).
    Step 5: Pre-render room: renderState.current.offscreenCanvas = preRenderRoom(grid, renderState.current.cameraOrigin, canvas.width, canvas.height, window.devicePixelRatio || 1).
    Step 6: Set runningRef.current = true.
    Step 7: Define frame():
      - FIRST LINE: if (!runningRef.current) return  — StrictMode guard
      - Get ctx and offscreen from renderState.current
      - Clear canvas: ctx.clearRect(0, 0, canvas.offsetWidth, canvas.offsetHeight)
      - Blit: ctx.drawImage(offscreen, 0, 0)  — single drawImage per frame (ROOM-08)
      - Schedule next: rafIdRef.current = requestAnimationFrame(frame)
    Step 8: rafIdRef.current = requestAnimationFrame(frame).
    Step 9: Return cleanup function:
      - runningRef.current = false  — MUST happen BEFORE cancelAnimationFrame
      - cancelAnimationFrame(rafIdRef.current)

    JSX return: a canvas element with ref={canvasRef} and style={{ width: '100%', height: '100%', display: 'block' }}.

    Critical implementation details:
    - if (!runningRef.current) return must be the FIRST statement in frame() — before any ctx access
    - runningRef.current = false must be set BEFORE cancelAnimationFrame in cleanup
    - clearRect uses CSS pixel dimensions (canvas.offsetWidth/Height), NOT physical pixel dimensions — ctx.scale handles the mapping
    - Do NOT use useState for any value read inside the frame() function
    - drawImage(offscreen, 0, 0) — integer coords only; camera offset is baked into OffscreenCanvas content
    - useEffect dependency array is [heightmap] only
  </action>
