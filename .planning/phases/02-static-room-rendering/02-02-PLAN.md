<task type="auto">
  <name>Task 1: Implement isoTileRenderer.ts — canvas drawing functions</name>
  <files>src/isoTileRenderer.ts</files>
  <action>
    Create `src/isoTileRenderer.ts` as a pure TypeScript module. No React imports. No default HSB colour — caller always provides one.

    Exports required:

    ```
    export const WALL_HEIGHT = 128; // 4 * TILE_H — named constant for all wall strip height draws
    ```

    `initCanvas(canvas: HTMLCanvasElement): CanvasRenderingContext2D`
    - Read `window.devicePixelRatio || 1` into dpr
    - Set `canvas.width = Math.floor(canvas.offsetWidth * dpr)`
    - Set `canvas.height = Math.floor(canvas.offsetHeight * dpr)`
    - Set `canvas.style.width = canvas.offsetWidth + 'px'`
    - Set `canvas.style.height = canvas.offsetHeight + 'px'`
    - Call `ctx.scale(dpr, dpr)`
    - Set `ctx.imageSmoothingEnabled = false`
    - Return ctx

    `computeCameraOrigin(grid: TileGrid, canvasCssWidth: number, canvasCssHeight: number): { x: number; y: number }`
    - Iterate all non-void tiles, call `tileToScreen(tx, ty, 0)` for each
    - Track minSx (sx - TILE_W_HALF), maxSx (sx + TILE_W_HALF), minSy (sy), maxSy (sy + TILE_H)
    - roomW = maxSx - minSx, roomH = maxSy - minSy (add WALL_HEIGHT to roomH to account for tallest wall)
    - Return `{ x: Math.floor((canvasCssWidth - roomW) / 2) - minSx, y: Math.floor((canvasCssHeight - roomH) / 2) - minSy }`
    - If grid has no non-void tiles, return `{ x: 0, y: 0 }`

    `preRenderRoom(grid: TileGrid, cameraOrigin: { x: number; y: number }, physicalW: number, physicalH: number, dpr: number, defaultHsb?: HsbColor): OffscreenCanvas`
    - Create `new OffscreenCanvas(physicalW, physicalH)`
    - Get context: `offscreen.getContext('2d')!`
    - Set `ctx.scale(dpr, dpr)` and `ctx.imageSmoothingEnabled = false`
    - Build a Renderable[] array — one entry per non-void tile:
      - Each renderable has tileX=tx, tileY=ty, tileZ=tile.height
      - Its draw function: computes `{ x: sx, y: sy } = tileToScreen(tx, ty, tile.height)`, adds cameraOrigin; calls `drawFloorTile` (top face rhombus), then conditionally `drawLeftFace` and `drawRightFace` for wall strips (see conditions below)
    - Call `depthSort(renderables)` — sort back-to-front
    - Iterate sorted renderables and call `renderable.draw(ctx)` for each
    - Return the OffscreenCanvas

    Internal drawing helpers (not exported — module-private):

    `drawFloorTile(ctx, sx, sy, hsb: HsbColor)`:
    - Get `{ top }` from `tileColors(hsb)` — only the top face fill
    - Draw rhombus path:
      - moveTo(sx, sy)                         // top vertex
      - lineTo(sx + TILE_W_HALF, sy + TILE_H_HALF) // right vertex
      - lineTo(sx, sy + TILE_H)                // bottom vertex
      - lineTo(sx - TILE_W_HALF, sy + TILE_H_HALF) // left vertex
      - closePath(), fillStyle = top, fill()

    `drawLeftFace(ctx, sx, sy, hsb: HsbColor)`:
    - Get `{ left }` from `tileColors(hsb)`
    - Left wall parallelogram (hangs below the left edge of the top rhombus):
      - moveTo(sx - TILE_W_HALF, sy + TILE_H_HALF)        // left vertex of rhombus
      - lineTo(sx, sy + TILE_H)                            // bottom vertex of rhombus
      - lineTo(sx, sy + TILE_H + WALL_HEIGHT)              // bottom-left of wall strip
      - lineTo(sx - TILE_W_HALF, sy + TILE_H_HALF + WALL_HEIGHT) // top-left of wall strip
      - closePath(), fillStyle = left, fill()

    `drawRightFace(ctx, sx, sy, hsb: HsbColor)`:
    - Get `{ right }` from `tileColors(hsb)`
    - Right wall parallelogram (hangs below the right edge of the top rhombus):
      - moveTo(sx + TILE_W_HALF, sy + TILE_H_HALF)         // right vertex of rhombus
      - lineTo(sx, sy + TILE_H)                             // bottom vertex of rhombus
      - lineTo(sx, sy + TILE_H + WALL_HEIGHT)               // bottom-right of wall strip
      - lineTo(sx + TILE_W_HALF, sy + TILE_H_HALF + WALL_HEIGHT) // top-right of wall strip
      - closePath(), fillStyle = right, fill()

    Wall strip draw conditions (evaluated inside each renderable's draw function, before drawing):
    - Left wall strip: `tx === 0 || grid.tiles[ty][tx - 1] == null` (loose equality catches undefined)
    - Right wall strip: `ty === 0 || grid.tiles[ty - 1]?.[tx] == null`

    Default HSB colour for tiles that have no per-tile colour set: `{ h: 220, s: 25, b: 80 }` (Habbo-style blue-grey floor). The `defaultHsb` parameter to `preRenderRoom` overrides this. In Phase 2, all tiles use the default unless the caller provides per-tile HSB — since the layout editor data model integration comes in Phase 4+, use default for all tiles now.

    Anti-patterns to avoid:
    - Do NOT import React, JSX, or any DOM API (document, window) at module level — keep pure for testability
    - Do NOT re-draw tiles every frame — preRenderRoom is called ONCE per layout change; the OffscreenCanvas is stored and blitted by RoomCanvas.tsx
    - Do NOT use RGB arithmetic for shading — use `tileColors()` from isoTypes.ts (HSL-based)
    - Do NOT set imageSmoothingEnabled only on the main canvas — ALSO set it on the OffscreenCanvas context
    - Wall detection: use loose == null (catches both null and undefined) not strict === null
