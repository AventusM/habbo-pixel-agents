  <name>Task 1: Integrate classification into AgentManager.trackAgent</name>
  <files>src/agentManager.ts, src/transcriptParser.ts, src/agentTypes.ts</files>
  <action>
    1. Update `src/agentTypes.ts`:
       - Add fields to `AgentState`: `role?: string`, `team?: TeamSection`, `taskArea?: string`
       - Import `TeamSection` from agentClassifier (or define inline to avoid circular deps)
       - Add new `ExtensionMessage` variants:
         - `{ type: 'agentCreated'; agentId: string; terminalName: string; variant: 0|1|2|3|4|5; role?: string; team?: TeamSection; taskArea?: string }`
         - `{ type: 'requestClassification'; agentId: string }` — sent when subagent_type is missing, triggers VS Code input prompt

    2. Update `src/agentManager.ts`:
       - Import `extractSubagentType`, `classifyAgent` from `agentClassifier.ts`
       - In `trackAgent()`, after creating the agent:
         a. Read the first ~50 lines of the JSONL file (using fs.readFileSync, not the watcher)
         b. Call `extractSubagentType(content)` to get subagent_type
         c. Call `classifyAgent(subagentType, content)` to get classification
         d. Set `state.role = classification.role`, `state.team = classification.team`, `state.taskArea = classification.taskArea`
         e. Set `terminalName = classification.displayName` (replaces "Claude N")
         f. If subagentType is null: also send `{ type: 'requestClassification', agentId }` to trigger user popup
       - Update `agentCreated` message to include role, team, taskArea
       - Add `reassignAgent(agentId: string, team: TeamSection): void` method that updates the agent's team and re-sends agentCreated

    3. Update `src/transcriptParser.ts`:
       - Add `extractSubagentTypeFromLine(line: string): string | null` — extracts subagent_type from a single JSONL line with Agent tool_use. This is a lighter version of agentClassifier's extractSubagentType, working on individual lines for the watcher.
       - Keep parseTranscriptLine unchanged — it handles the streaming watch events.
  </action>
  <verify>npx vitest run --reporter=verbose</verify>
  <done>AgentManager classifies agents on discovery, sends role/team data to webview, triggers prompt for unclassified agents</done>
