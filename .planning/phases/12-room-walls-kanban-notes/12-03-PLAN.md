  <action>
Create `src/isoKanbanRenderer.ts`:

1. **Status color mapping:**
   ```typescript
   const KANBAN_COLORS: Record<string, string> = {
     'Todo': '#fef08a',
     'In Progress': '#93c5fd',
     'Done': '#86efac',
     'No Status': '#e5e7eb',
   };

   export function statusToColor(status: string): string {
     return KANBAN_COLORS[status] ?? '#e5e7eb';
   }
   ```

2. **Wall note position functions:**
   - `leftWallNotePosition(ty: number, cameraOrigin: { x: number; y: number }): { x: number; y: number }`:
     - Compute `tileToScreen(0, ty, 0)` for the left wall tile.
     - Return x shifted left onto the wall face: `sx + cameraOrigin.x - TILE_W_HALF * 0.5`
     - Return y at ~40% down the wall: `sy + cameraOrigin.y + TILE_H_HALF + WALL_HEIGHT * 0.4`
   - `rightWallNotePosition(tx: number, cameraOrigin: { x: number; y: number }): { x: number; y: number }`:
     - Compute `tileToScreen(tx, 0, 0)`.
     - Return x shifted right onto the wall face: `sx + cameraOrigin.x + TILE_W_HALF * 0.5`
     - Return y at ~40% down the wall: `sy + cameraOrigin.y + TILE_H_HALF + WALL_HEIGHT * 0.4`

3. **Single sticky note draw function:**
   ```typescript
   function drawStickyNote(
     ctx: CanvasRenderingContext2D,
     screenX: number,
     screenY: number,
     title: string,
     color: string,
   ): void
   ```
   - Draw a filled rounded rectangle (48x36 px) centered at `(screenX, screenY)` with 3px border radius.
   - Fill with `color`, stroke with `rgba(0,0,0,0.6)` 1px border.
   - Draw title text in 5px "Press Start 2P" font, dark color `#1a1a2e`, centered.
   - Truncate title to 10 chars + ellipsis if longer.

4. **Main export function:**
   ```typescript
   export function drawKanbanNotes(
     ctx: CanvasRenderingContext2D,
     cards: KanbanCard[],
     grid: TileGrid,
     cameraOrigin: { x: number; y: number },
   ): void
   ```
   - If `cards.length === 0`, return immediately (no notes to draw).
   - Determine available wall slots:
     - Left wall: count non-void tiles in column 0 (tiles where `grid.tiles[ty][0] !== null`). Each tile holds up to 2 notes (one at ~30% wall height, one at ~60%).
     - Right wall: count non-void tiles in row 0 (tiles where `grid.tiles[0][tx] !== null`). Same 2-per-tile capacity.
   - Total capacity = `(leftWallTiles + rightWallTiles) * 2`.
   - Take at most `capacity` cards from the array.
   - Distribute cards: fill left wall first (top to bottom, 2 per tile row), then right wall (left to right, 2 per tile column).
   - For each card, compute screen position using `leftWallNotePosition`/`rightWallNotePosition` with a vertical offset for the second note in a tile (shift down ~40px for the second note).
   - Call `drawStickyNote` for each card.

Create `tests/isoKanbanRenderer.test.ts`:
- Test `statusToColor` returns correct hex for 'Todo', 'In Progress', 'Done', 'No Status'.
- Test `statusToColor` returns grey for unknown status string.
- Test `drawKanbanNotes` with empty cards array does not throw.
- Test `drawKanbanNotes` with 3 cards on a 5x5 grid positions notes on the wall.
