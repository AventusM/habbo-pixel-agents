```typescript
  bookshelf: { widthTiles: 2, heightTiles: 1 },
  rug: { widthTiles: 1, heightTiles: 1 },
  whiteboard: { widthTiles: 1, heightTiles: 1 },
};
```

**placeFurniture(grid: TileGrid, furnitureList: FurnitureSpec[], multiTileFurnitureList: MultiTileFurnitureSpec[], tileX: number, tileY: number, furnitureType: string, direction: number): boolean**
- Get spec from FURNITURE_SPECS[furnitureType]
- Validate footprint: loop through widthTiles × heightTiles
  - Check each tile (tileX+dx, tileY+dy) is within grid bounds
  - Check each tile is walkable (not null in grid.tiles)
  - If any tile invalid, return false (placement rejected)
- Create FurnitureSpec or MultiTileFurnitureSpec object
- Get tileZ from grid.tiles[tileY][tileX].height
- Push to appropriate list (furnitureList for 1×1, multiTileFurnitureList for multi-tile)
- Return true (placement succeeded)

**rotateFurniture(currentDirection: number): number**
- Habbo rotations: [0, 2, 4, 6] (NE, SE, SW, NW)
- Find current index in array, return (index + 1) % 4
- Return new direction value

**saveLayout(grid: TileGrid, tileColorMap: Map&lt;string, HsbColor&gt;, furniture: FurnitureSpec[], multiTileFurniture: MultiTileFurnitureSpec[], doorCoords: {x, y, z, dir}): string**
- Create LayoutData object: { heightmap: gridToHeightmap(grid), doorX, doorY, doorZ, doorDir, tileColors: Object.fromEntries(tileColorMap), furniture, multiTileFurniture }
- Return JSON.stringify(data, null, 2)

**loadLayout(jsonString: string): LayoutData**
- Parse JSON string
- Return LayoutData object with all fields
- Caller is responsible for updating grid, tileColorMap, furniture lists, door coords

**Update EditorState interface:**
Add selectedFurniture and furnitureDirection fields.

- saveLayout/loadLayout: round-trip preserves all data
