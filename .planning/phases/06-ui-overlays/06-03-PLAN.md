   - Generate font URI: `const fontUri = panel.webview.asWebviewUri(vscode.Uri.joinPath(context.extensionUri, 'dist', 'webview-assets', 'PressStart2P-Regular.ttf'));`
   - Update CSP meta tag: Add `font-src ${panel.webview.cspSource};` to content attribute
   - Add <link rel="preload"> in <head> BEFORE <style>: `<link rel="preload" href="${fontUri}" as="font" type="font/ttf" crossorigin />`
   - Add @font-face declaration in <style> tag:
   ```css
   @font-face {
     font-family: 'Press Start 2P';
     src: url('${fontUri}') format('truetype');
     font-weight: normal;
     font-style: normal;
     font-display: block; /* Blocks text render until font loads - prevents FOUT */
   }
   ```

4. **Existing code already sets ctx.font = '8px "Press Start 2P"'** in isoBubbleRenderer.ts and isoNameTagRenderer.ts — no changes needed.

CRITICAL:
- Use font-display: block (NOT swap or auto) to prevent FOUT — text will be invisible for 100-300ms until font loads, but no jarring font switch
- Include crossorigin attribute on <link rel="preload"> even though font is same-origin (required by spec)
- Add font-src to CSP or font loading will fail with CSP violation

NOTES:
- Press Start 2P is OFL 1.1 licensed (open source, safe for bundling)
- Volter font (UI-08) deferred to future — requires extension settings UI for opt-in and licensing disclaimer
- Font file size ~40KB (negligible impact on extension bundle size)
