   - text: string (agent log line or tool name)
   - anchorX: number (avatar head screen X position)
   - anchorY: number (avatar head screen Y position - top of avatar sprite)
   - maxWidth?: number (default 200px for word wrap)
   - isWaiting?: boolean (true for animated "..." bubble)

2. **wrapText function:**
   - Split text by spaces (word-by-word, NOT character-by-character)
   - Accumulate words into lines using ctx.measureText() width checks
   - When measureText(testLine).width > maxWidth, push current line and start new line with overflowing word
   - Return string[] (array of wrapped lines)
   - Edge case: If single word exceeds maxWidth, allow overflow (do NOT truncate)

3. **drawSpeechBubble function:**
   - Set ctx.font = '8px "Press Start 2P"' BEFORE measureText calls (measurement depends on font)
   - If isWaiting=true, override text with cycling dots: '.repeat(Math.floor((currentTimeMs % 1500) / 500) + 1)' (1-3 dots)
   - Call wrapText to get lines array
   - Calculate bubble dimensions: width = max(textWidths) + padding*2, height = lines.length * lineHeight + padding*2
   - Position bubble centered above anchor: bubbleX = anchorX - width/2, bubbleY = anchorY - height - tailHeight - 4px gap
   - Draw downward-pointing triangle tail: moveTo(anchorX, anchorY-4) → lineTo(anchorX-5, bubbleY+height) → lineTo(anchorX+5, bubbleY+height) → closePath, fill white
   - Draw rounded rectangle background: ctx.roundRect(bubbleX, bubbleY, width, height, 8px radius), fill white
   - Draw border: same roundRect with ctx.stroke (2px black stroke)
   - Draw text lines: for each line, ctx.fillText(line, Math.floor(bubbleX+padding), Math.floor(bubbleY+padding+i*lineHeight)) — use Math.floor() to avoid sub-pixel blur

Constants: padding=8, lineHeight=16, tailHeight=8, cornerRadius=8

```typescript
// Render speech bubbles (always on top of avatars)
for (const avatar of renderState.current.avatars) {
  const { x: screenX, y: screenY } = tileToScreen(avatar.tileX, avatar.tileY, avatar.tileZ);
  const headY = screenY - AVATAR_HEIGHT; // Avatar head position (top of sprite)

  // Demo text: show avatar ID and state
  const text = `${avatar.id}: ${avatar.state}`;

  ctx.save();
  ctx.translate(renderState.current.cameraOrigin.x, renderState.current.cameraOrigin.y);
  drawSpeechBubble(ctx, {
    text,
    anchorX: screenX,
    anchorY: headY,
    isWaiting: avatar.state === 'idle', // Demo: idle avatars show waiting bubble
  });
  ctx.restore();
}
```
