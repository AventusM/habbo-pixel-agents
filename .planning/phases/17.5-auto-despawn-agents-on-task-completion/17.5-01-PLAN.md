1. Add a constant `COMPLETION_STALENESS_MS = 15_000` (15 seconds) at the top of `src/agentManager.ts` next to the other timeout constants.

2. Add a private method `isAgentCompleted(jsonlPath: string): boolean` to `AgentManager` that:
   - Gets file size via `fs.statSync`. Returns false if size is 0.
   - Reads the last 8192 bytes (or entire file if smaller) using `fs.openSync`/`fs.readSync`/`fs.closeSync` with a position of `stat.size - READ_SIZE`.
   - Splits by newline, filters empty lines, takes the last line.
   - Parses as JSON.
   - Returns `true` if `entry.type === 'assistant'` AND `entry.message?.stop_reason === 'end_turn'`.
   - Wraps everything in try/catch, returns `false` on any error (parse failure, file access, etc).

3. Modify `checkIdleAgents()` to add completion detection AFTER the file-existence check and BEFORE the idle-timeout check:
   - Get `fs.statSync(jsonlPath)` and compute `staleness = now - stat.mtimeMs`.
   - If `staleness > COMPLETION_STALENESS_MS`, call `this.isAgentCompleted(jsonlPath)`.
   - If completed, log `[AgentManager] Agent completed (end_turn + Xs stale): {agentId}` and push to `toRemove`.
   - Wrap the stat+completion block in try/catch (handles race with file deletion between existsSync and statSync).
