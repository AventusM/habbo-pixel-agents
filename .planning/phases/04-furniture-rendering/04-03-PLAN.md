2. Define test furniture layout (8 furniture types in office layout):
```typescript
const furnitureList: FurnitureSpec[] = [
  { name: 'chair', tileX: 3, tileY: 3, tileZ: 0, direction: 0 },
  { name: 'chair', tileX: 5, tileY: 3, tileZ: 0, direction: 2 },
  { name: 'lamp', tileX: 7, tileY: 2, tileZ: 0, direction: 0 },
  { name: 'plant', tileX: 1, tileY: 5, tileZ: 0, direction: 0 },
  { name: 'computer', tileX: 4, tileY: 4, tileZ: 1, direction: 0 }, // On desk height
  // Add remaining 3 types (rug, whiteboard, bookshelf) as identified in FURNITURE_VALIDATION.md
];

const multiTileFurniture: MultiTileFurnitureSpec[] = [
  { name: 'desk', tileX: 4, tileY: 3, tileZ: 0, widthTiles: 2, heightTiles: 1, direction: 0 },
  // Add bookshelf, rug if multi-tile
];
```

3. Insert furniture into depth-sort pipeline (in render loop, after floor tiles):
```typescript
// Existing floor tile rendering...
for (const tile of floorTiles) {
  renderables.push(tile);
}

// NEW: Add furniture renderables
for (const furni of furnitureList) {
  const renderable = createFurnitureRenderable(furni, spriteCache, 'furniture');
  renderables.push(renderable);
}

for (const furni of multiTileFurniture) {
  const renderable = createMultiTileFurnitureRenderable(furni, spriteCache, 'furniture');
  renderables.push(renderable);
}

// Depth sort all renderables (tiles + furniture)
const sorted = depthSort(renderables);

// Draw in back-to-front order
for (const renderable of sorted) {
  renderable.draw(ctx);
}
```

4. **CRITICAL:** Ensure furniture renderables are added BEFORE depthSort call, not after. Furniture must be depth-sorted WITH tiles, not drawn in a separate pass.

5. Use actual furniture names from FURNITURE_VALIDATION.md. If a furniture type has placeholder sprites, comment it out or use 'chair' as fallback.

**DO NOT** add any React state for furniture — use static const arrays for now. Editing will come in Phase 7.

**DO NOT** implement furniture placement UI — this task is render-only.
